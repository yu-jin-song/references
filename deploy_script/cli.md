# 유용한 명령어

### 1. bosh cancel-task
```shell
bosh cancel-task [ID]
```

### 2. cf ssh
```shell
cf ssh <APP_NAME> [--process PROCESS] [-i INDEX] [-c COMMAND]
```

### 3. cf scale
```shell
cf scale <APP_NAME> [-i INSTANCES] [-k DISK] [-m MEMORY] [-f]
```

### 4. cf logs
```shell
cf logs <APP_NAME> [--recent]
```
